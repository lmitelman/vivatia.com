THUBAN SOFTWARE

GESTOR DOCUMENTAL

Plataforma diseñada para la creación de soluciones donde los documentos y procesos son la clave del negocio.

Seguridad


Thuban se conecta a la base de datos mediante un usuario y clave que se configura en un archivo de seguridad. Dicho archivo puede configurarse para que se encripte al iniciar la aplicación por primera vez.

Historial


Una función que tiene Thuban Software que resulta muy útil a la hora de auditar documentos, es el visor de la historia del documento. Cada documento cuenta con un módulo de auditoria que permite ver las acciones realizadas sobre él. El mismo registra cuando y por quien un documento fue creado, consultado, editado y cuáles fueron las modificaciones realizadas.

Notificaciones


Además cuenta con un módulo de envío automático de notificaciones por correo electrónico el cual se puede configurar en distintas etapas del flujo de trabajo y puede ser consecuencia de, por ejemplo, un cambio de estado de un documento o una alerta a una regla de negocio.

Monitores


También vas a contar con herramientas para poder monitorear el proceso en línea, efectuar seguimientos y poder actuar en función de la información que dichos monitores indiquen.

Grupos y permisos


Con Thuban vas a poder crear perfiles, grupos y usuarios a un alto nivel de detalle y asignación de permisos. Estos serán definidos de acuerdo con las necesidades de la solución.
